# E-commerce Returns Data Analysis Project

## Overview
This project analyzes e-commerce product return behavior using SQL, Tableau, and dataset processing techniques. The goal is to understand return patterns, customer behavior, and logistics impact.

## Tools & Technologies
- SQL Server
- Tableau
- CSV Dataset
- Data Visualization

## Files Description
- Dashboard_Midterm.twbx: Tableau dashboard file
- Dataset (1).csv: Raw dataset used for analysis
- SQLQuery_TruyVan.sql: SQL queries for data analysis
- dashboard_image.png: Visualization preview of dashboard

## Key Analysis
- Return behavior trends
- Customer segmentation
- Logistics-related return issues
- Time-based return patterns
